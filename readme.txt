
---- Краткое руководство игры -----

1. При попадании вражеского лазера в корабль игрока отнимается - 10 здоровья.
2. При прихождении вражеского корабля до низа экрана отнимается 1 жизнь.
3. Новый уровень начинается при убийвстве всех врагов.
4. При уровне здоровья 0 и 0 жизней, игра проигрывается и перезапускается.
5. Передвигаться - W, A, S, D ; стрелять - space. 
6. Враги умирают от одного попадания, или столкновения(корабль игрока так же получает урон).
7. Удачной игры!
